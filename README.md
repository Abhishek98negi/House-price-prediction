# 🏠 House Price Prediction

A machine learning project that predicts house prices based on various housing features such as area, number of bedrooms, bathrooms, and location. This project uses regression algorithms to estimate house prices accurately.

## 📌 Project Overview

House price prediction is one of the most common regression problems in machine learning. The value of a house depends on multiple factors such as location, size, number of rooms, and other facilities. In this project, machine learning models are trained to analyze these factors and predict the price of a house.

The goal is to build an efficient model that can help buyers, sellers, and real estate businesses make informed decisions.

## 📂 Dataset

The dataset used for this project contains house-related features such as:

* **Area (sq ft)** – Total size of the house
* **Bedrooms** – Number of bedrooms
* **Bathrooms** – Number of bathrooms
* **Floors** – Number of floors
* **Location** – Area/City of the property
* **Price** – Target variable

The dataset is used to train regression models for price prediction.

---

## ⚙️ Workflow

1. Import and load the dataset
2. Perform data cleaning and preprocessing
3. Analyze important features
4. Visualize relationships between variables
5. Split the dataset into training and testing sets
6. Train regression models
7. Evaluate model performance
8. Predict house prices

---

## 🤖 Machine Learning Models Used

## 📊 Evaluation Metrics

To evaluate the performance of the model:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* Root Mean Squared Error (RMSE)
* R² Score

These metrics help measure how accurate the predictions are.
